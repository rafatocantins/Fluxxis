# UI Generation Skill

## Purpose
Generate animated, interactive UI components using AI-powered tools and component libraries.

## Capabilities
- Generate components from natural language descriptions
- Access 185+ animated component templates
- Create custom variants with brand alignment
- Implement scroll-based and hover animations
- Build responsive layouts

## When to Use
- Rapid prototyping of new components
- Generating animation variants
- Creating marketing/conversion UIs
- Building interactive demos

## MCP Integration
- `magicuidesign-mcp` — 50+ Magic UI components
- `reactbits` — 135+ ReactBits components
- `21st-dev-magic` — AI-driven generation
- `figma` — Design token sync
