# Component Builder Skill

## Purpose
Build intent-driven, goal-first React components with adaptive behavior for the AI Design System.

## Capabilities
- Create SmartCTA, SmartSection, SmartLayout components
- Implement goal declaration contracts
- Add behavioral observers (scroll, hover, dwell)
- Build fallback mechanisms for offline/API failure
- Ensure WCAG 2.1 AA accessibility compliance

## When to Use
- Building new smart components
- Modifying component contracts
- Adding animation variants
- Implementing intent tokens

## MCP Integration
- `magicuidesign-mcp` — Animated buttons, text effects, backgrounds
- `reactbits` — 135+ animated component templates
- `21st-dev-magic` — AI-driven component generation
- `figma` — Design token extraction

## Output
- TypeScript/TSX with proper typing
- Clear prop interfaces
- Usage examples
- Inline JSDoc for complex logic
