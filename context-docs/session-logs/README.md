# Session Logs — AI Design System

This folder contains logs of development sessions, including context, decisions, and outcomes.

## Structure

```
session-logs/
├── YYYY-MM-DD-session-name.md
└── README.md
```

## Session Guidelines

1. **Log every significant session** - Any work >30 minutes or involving multiple decisions
2. **Capture decisions and rationale** - Future context for why choices were made
3. **Link to task records** - Cross-reference with detailed task documentation
4. **Note blockers immediately** - Don't wait until session end to document issues
5. **Define next steps clearly** - Make handoff to next session seamless
